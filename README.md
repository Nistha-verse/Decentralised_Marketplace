# Decentralised_Marketplace
# 🛒 Decentralized Marketplace on Stellar (Soroban)

## 📌 Project Description

This project is a basic decentralized marketplace smart contract built using Soroban on the Stellar network. It allows users to list items for sale and purchase them in a trustless environment without intermediaries.

The contract demonstrates core Web3 marketplace concepts such as ownership, listings, and transaction validation using Stellar's smart contract platform.

---

## ⚙️ What it does

The smart contract enables:

- Sellers to list items with a name and price
- Buyers to purchase available items
- Storage and retrieval of marketplace listings on-chain
- Basic ownership and sale status tracking

---

## ✨ Features

- 📦 List Items  
  Users can create marketplace listings with price and metadata

- 💰 Buy Items  
  Buyers can purchase listed items (mock logic — token transfer can be added)

- 🔍 View Listings  
  Retrieve individual items or all marketplace listings

- 🔐 Authorization  
  Ensures only the seller can list items and buyers must approve purchases

- 🧱 On-chain Storage  
  All listings are stored securely on the Stellar blockchain via Soroban
